# OnlineExam
Online portal for coding examination 

For Services.:
                
1.	git clone https://github.com/axd164330/OnlineExam.git
2.	cd services 
3.	mvn install
4.	then deploy WAR on tomcat


For Front end:
1.	git clone https://github.com/axd164330/OnlineExam.git
2.	cd Front end
3.	npm install
4.	npm start
